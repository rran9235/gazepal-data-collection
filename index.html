GazePal Data Collection

Instructions:

  1. Read these instructions before clicking "Next".
  2. Fill in the form and click "Submit". The form collects information only to help organise the data.
  3. You will be shown a visual marker to fixate upon for the recording.
  4. You will need to gaze at the marker for a 10 second period without distraction.
  5. The process will be repeated as the marker moves to various screen regions.
  6. The output will generate several video (.mp4) and text (.txt) files prefixed with "GazePal".
  7. Please share these videos with me so I can improve my machine learning algorithm
